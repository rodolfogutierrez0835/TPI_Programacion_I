## Descripción del proyecto

El presente trabajo consiste en el desarrollo de una aplicación de consola en Python para la gestión de datos de países.

El sistema permite administrar información almacenada en un archivo CSV, realizando operaciones de consulta, modificación y actualización de los datos de manera segura mediante validaciones y manejo de excepciones.

## Características del sistema

El sistema fue diseñado siguiendo principios de programación estructurada, incorporando:

- Modularización mediante funciones.
- Persistencia de datos mediante archivos CSV.
- Validación de entradas del usuario.
- Manejo de excepciones mediante try/except.
- Algoritmos de búsqueda y ordenamiento.
- Operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre los datos almacenados.

## Funcionalidades

El sistema permite:

- Mostrar países.

- Agregar nuevos países.

-  Actualizar la información de un país.

-  Buscar países por nombre.

-  Filtrar países por continente.

- Ordenar países según distintos criterios.

- Mostrar estadísticas generales.

- Guardar los cambios realizados en el archivo CSV.

- Eliminar países del sistema.

## Validaciones implementadas

El programa incorpora diversas validaciones para evitar errores de ejecución:

- Validación de datos numéricos.
- Validación de nombres vacíos.
- Comparaciones sin distinguir mayúsculas y minúsculas.
- Validación de continentes permitidos.
- Manejo de excepciones mediante try/except.
- Verificación de existencia de registros.
- Control de errores en la lectura del archivo CSV.
